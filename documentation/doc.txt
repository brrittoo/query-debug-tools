Local Development (From Git Clone)

Clone outside your Laravel project:

git clone https://github.com/brrittoo/laravel-url-encoder.git ../debug-tools


Add as a path repository:

"repositories": [
    {
        "type": "path",
        "url": "../debug-tools",
        "options": { "symlink": true }
    }
]


Require locally:
composer require brrittoo/url-encoder:@dev
Publish configuration:

php artisan vendor:publish --provider="Brrittoo\UrlEncoder\UrlEncoderServiceProvider" --tag="url-encoder-config"





âœ… Symlinks the clone; changes reflect immediately.
